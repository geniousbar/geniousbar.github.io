### 数据仓库书籍推荐
```
BI学习书籍推荐

看了好几个园子里的朋友的回复，觉得有不少朋友对BI和数据仓库的概念还没有一个比较完整的了解。不过能够来到这里作客真的非常欢迎。如果有兴趣一起研究BI的朋友，可以看看以下我推荐的书籍，以下的书籍我都买过了，都看过，但是有的只是粗粗看了一遍。

1.《数据仓库》W.H.Inmon著，王志海 等译（注：这本书可以说是我的入门书，堪称经典，就是比较枯燥）

2.《数据仓库设计》Claudia Imhoff,Nicholas Galemmo，Jonathan G.Geiger著，于戈，鲍玉斌，王大玲 等译（注：感觉不错的一本书，主要介绍业务建模的方法，比较深入）

3.《电子商务中的数据仓库技术》W.H.Inmon，R.H.Terdeman,Joyee Norris-Montanari,Dan Meers著，张铭，金彦钟 等译（注：这本书我原来是为了应付面试买的，但是后来没有用到，感觉讲的东西都是点到为止，不够深入）

4.《Sqlserver2005 Analysis Services 标准指南》Edward Melomed,Irina Gorbach,Alexander Berger,Py Bateman 著，武桂香，孙赫雄，杜宏旺译（注：工具书，学习Sqlserver2005 Analysis Services 的书，感觉和MSDN查不多，看起来方便一点，然后加了一些扩展）

5.《Sqlserver2005 Integration Services 标准指南》BillBaker，Kirk Haselden著,马树奇，金燕译(注：同上)

6.《Sqlserver2005 Reporting Services 标准指南》Michael Lisin,Jim Joseph著,路晓村，王杰，高春蓉译(注：同上)

7.《数据仓库项目管理实践》于宗民，刘义宁，祁国辉 著(注:这本书主要讲数据仓库应用项目开发过程的管理的，里面提供了很多现成的文档的模板，可以拿过来借鉴一下)

8.《数据仓库工具箱：面向SQL Server 2005和Microsoft商业智能工具集》 （美）曼蒂，（美）桑斯维特，（美）金伯尔　著，闫雷鸣，冯飞　译（注：好书，比较全面的一本好书）

9.《数据挖掘原理与应用：SQL Server 2005数据库》（注：专门讲数据挖掘的一本比较深入的书，好书，个人比较喜欢,正在研究中）


以上这些书，基本上都是我这2007年5月份以后买的书，有几本好书值的一看，但是其他的书也不算差，呵呵，我买的时候，都是先看目录和看网友评价的
```
